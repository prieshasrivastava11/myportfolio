TYPEFACE & PACKAGING DESIGN

About typeface:

This typeface has been designed in a way that depicts my personality of being stubborn but still flexible. Every letter has been designed using rounded triangles as the negative space to create each letter out of a rectangle. Key features
Ratio- 4:5
Rounded triangle of same ratio of height 3 and base 2 is used to create negative space
Width of 1 box is taken in all letters
All corners are slightly rounded
